# telegram_auth

### Installation

Clone project:

````shell
git clone https://github.com/ruslan-kornich/telegram_auth.git
````
Create venv:
````shell
python -m venv venv
````

Activate venv:
```shell
source venv\bin\activate 
```

Install requirements:
```shell
pip install -r requirements.txt
```
Create .env file 

Run migration:
```shell
python manage.py migrate
```

Run bot and server.
